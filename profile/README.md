## Using the packages

To use a package, first add the the group's local registry `CompBayesRegistry`. 
In Julia package mode (press `]` to enter that mode) and run:

```registry add https://github.com/compbayes/CompBayesRegistry.git```

Then add a package, again in Julia package mode:

```add PackageName```
